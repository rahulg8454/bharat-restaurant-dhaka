🍽️ Bharat Restaurant – Digital Menu

A modern, responsive digital restaurant menu web app with cart functionality, built using React and deployed on Vercel.

🔗 Live Demo:
👉 https://bharat-restaurant-dhaka.vercel.app/
🚀 Features

🧾 Complete digital menu with categories (Snacks, Chinese, Noodles, etc.)

🌐 Multi-language support (Hindi + English)

🛒 Cart system with:

Add / Remove items

Quantity management (+ / −)

Real-time total calculation

🍜 Option-based items (Half / Full, Cup / Cone, etc.)

📱 Fully responsive (mobile-first design)

🟢 Veg-only indicator system

⚡ Fast deployment using Vercel

🧠 Advanced Functionality

Dynamic item quantity tracking in menu

Swiggy-style UI (inline + / − buttons)

Option selection popup for customizable items

Smart cart ID handling for different item variants

Sticky cart bar with total items & price

🛠️ Tech Stack

Frontend: React + TypeScript

Styling: Tailwind CSS

Icons: Lucide React

State Management: Context API

Build Tool: Vite

Deployment: Vercel

⚙️ Installation & Setup
# Clone repository
git clone https://github.com/your-username/bharat-swad-menu.git

# Go to project folder
cd bharat-swad-menu

# Install dependencies
npm install

# Run locally
npm run dev
🌍 Deployment

Deployed using Vercel:

npm run build

Then connect GitHub repo to Vercel for automatic deployment.
